# 고도인재 포인트 평가 시스템 · Japan HSP Points Calculator

[简体中文](README.md) | [繁體中文](README.zh-TW.md) | [日本語](README.ja.md) | **한국어** | [English](README.en.md)

일본 고도인재 비자(고도전문직)의 포인트 자가진단 도구입니다. 출입국재류관리청 공식 포인트표(고도전문직 성령)에 따라 해당 항목을 선택하면 점수를 실시간으로 계산합니다. **70점**이면 고도인재 비자 요건을 충족하고, **80점**이면 최단 1년 만에 영주 신청이 가능합니다.

**온라인 체험: [point.keihan.co](https://point.keihan.co)** · © 京阪住業株式会社 KEIHAN JUGYO K.K.

## 기능

- **3개 분야**: 고도 학술연구(イ) / 고도 전문·기술(ロ) / 고도 경영·관리(ハ) — 분야별 독립된 포인트표와 선택 항목
- **완전한 포인트 규칙**: 학력, 경력, 연령, 연수입(연령 연동 구간 포함), 연구 실적, 지위, 일본어 능력, 특별 가점
- **5개 언어**: 중국어 간체 / 중국어 번체 / 일본어 / 한국어 / 영어 원클릭 전환
- **실시간 점수 패널**: 70 / 80점 달성 상태, 우대 혜택 목록, 진행 바
- **규칙 충돌 알림**: N2와 「일본 대학 졸업」 가점 병용 불가, 중소기업 가점은 이노베이션 지원조치와 함께 선택해야 유효 등
- **점수 향상 팁**: 목표 점수에 미달할 때 가장 효과적인 가점 항목을 자동 제시
- **라이트 / 다크 테마**: 원클릭 전환, 설정은 로컬에 저장
- **모바일 지원**: 반응형 레이아웃 + 하단 고정 점수 바

## 사용법

바로 체험하려면 설치 없이 **[point.keihan.co](https://point.keihan.co)** 로 이동하세요.

로컬 실행: 순수 정적 페이지로, 빌드나 런타임 의존성이 전혀 없습니다.

```bash
# 브라우저로 열기만 하면 됩니다
open index.html
```

또는 임의의 정적 호스팅(GitHub Pages, Netlify 등)에 배포하세요. 루트 디렉터리가 곧 사이트입니다.

```
├── index.html                 # 단일 파일 앱: 페이지·스타일·채점 로직
└── assets/
```

기술적으로는 자체 완결형 단일 HTML입니다. Keihan 디자인 시스템의 색상·타이포그래피 토큰을 인라인으로 포함하고, 채점 엔진과 다국어 렌더링을 순수 JavaScript로 구현했으며, 폰트는 Google Fonts에서 로드합니다(시스템 폰트 폴백 포함).

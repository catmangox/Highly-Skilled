# 高度人材ポイント計算 · Japan HSP Points Calculator

[简体中文](README.md) | [繁體中文](README.zh-TW.md) | **日本語** | [한국어](README.ko.md) | [English](README.en.md)

日本の高度人材ビザ（高度専門職）のポイント自己診断ツールです。出入国在留管理庁の公式ポイント表（高度専門職省令）に基づき、該当項目を選択するだけで合計点をリアルタイムに計算します。**70 点**で高度専門職に該当、**80 点**で永住申請が最短 1 年になります。

**オンライン体験：[point.keihan.co](https://point.keihan.co)** · © 京阪住業株式会社 KEIHAN JUGYO K.K.

## 機能

- **3 つの類型**：高度学術研究（イ）/ 高度専門・技術（ロ）/ 高度経営・管理（ハ）— 類型ごとに独立したポイント表と選択項目
- **完全なポイント規則**：学歴、職歴、年齢、年収（年齢に連動する区分を含む）、研究実績、地位、日本語能力、特別加算
- **5 言語対応**：簡体字中国語 / 繁体字中国語 / 日本語 / 韓国語 / 英語をワンクリックで切替
- **リアルタイム得点パネル**：70 / 80 点の達成状況、受けられる優遇の一覧、プログレスバー
- **規則の競合警告**：N2 と「本邦の大学卒業」の加点は併用不可、中小企業の加点はイノベーション支援措置とセットで有効、など
- **加点のヒント**：目標点に届かない場合、最も効果的な加点項目を自動で提示
- **ライト / ダークテーマ**：ワンクリックで切替、設定はローカルに保存
- **モバイル対応**：レスポンシブレイアウト + 画面下部の固定スコアバー

## 使い方

すぐに試したい方はインストール不要で **[point.keihan.co](https://point.keihan.co)** へ。

ローカル実行：純粋な静的ページのため、ビルドも実行時依存も不要です。

```bash
# ブラウザで開くだけ
open index.html
```

または任意の静的ホスティング（GitHub Pages、Netlify など）にデプロイしてください。ルートディレクトリがそのままサイトになります。

## プロジェクト構成

```
├── index.html                 # 単一ファイルアプリ：ページ・スタイル・採点ロジック
└── assets/
    └── logo-mark-zinc.png     # KEIHAN ブランドマーク（テーマに応じて自動反転）
```

技術的には自己完結の単一 HTML です。Keihan デザインシステムのカラー・タイポグラフィのトークンをインライン化し、採点エンジンと多言語レンダリングを素の JavaScript で実装、フォントは Google Fonts から読み込みます（システムフォントへのフォールバック付き）。

## 免責事項

本ツールは高度専門職省令のポイント表に基づく参考計算です。認定の可否は日本の出入国在留管理庁の審査によります。

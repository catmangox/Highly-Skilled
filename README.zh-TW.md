# 高度人才評分系統 · Japan HSP Points Calculator

[简体中文](README.md) | **繁體中文** | [日本語](README.ja.md) | [한국어](README.ko.md) | [English](README.en.md)

日本高度人才簽證（高度専門職）積分自測工具。依據出入國在留管理廳官方打分表（高度専門職省令），勾選自身條件即可即時計算得分：**70 分**即符合高度人才簽證要求，**80 分**最快 1 年可申請永住。

**線上體驗：[point.keihan.co](https://point.keihan.co)** · © 京阪住業株式會社 KEIHAN JUGYO K.K.

## 功能

- **三大類別**：高度學術研究（イ）/ 高度專業技術（ロ）/ 高度經營管理（ハ），各類別獨立打分表與選項
- **完整積分規則**：學歷、工作經歷、年齡、年收入（含按年齡連動的檔位）、研究實績、職位、日語能力、特別加分
- **五種語言**：簡體中文 / 繁體中文 / 日本語 / 한국어 / English，一鍵切換
- **即時得分面板**：70 / 80 分門檻狀態、可享待遇清單、進度條
- **規則衝突提示**：如 N2 與「日本大學畢業」加分不可併用、中小企業加分需搭配創新支援措施等
- **提升建議**：距離目標分數時，自動列出最有效的加分項
- **明暗主題**：一鍵切換 Light / Dark，偏好本地記憶
- **行動端適配**：響應式版面 + 底部懸浮得分欄

## 使用

想直接體驗？無需安裝，前往 **[point.keihan.co](https://point.keihan.co)** 即可。

本地執行：純靜態頁面，無需建置，無任何執行時依賴：

```bash
# 直接用瀏覽器開啟即可
open index.html
```

或部署到任意靜態託管（GitHub Pages、Netlify 等），根目錄即站點。

## 專案結構

```
├── index.html                 # 單檔應用：頁面、樣式、打分邏輯
└── assets/
    └── logo-mark-zinc.png     # KEIHAN 品牌標誌（隨主題自動反色）
```

技術上為單一自包含 HTML：內聯 Keihan 設計系統色彩與排版 token，原生 JavaScript 實現打分引擎與多語言渲染，字體透過 Google Fonts 載入（含系統字體回退）。
